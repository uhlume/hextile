    Global

    variables

    cell-width

    $cell-width: null !default;

    Description

    Fixed width for hex cells. If null (the default value), cells will automatically size themselves to fit available space in the container element. Accepts any CSS length or percentage value.

    cell-padding

    $cell-padding: 0 !default;

    Description

    Hex interior padding. All cells within a Hextile grid share a single uniform padding value. Accepts any CSS length or percentage value.

    (Note: This is a limitation of the technique Hextile uses to allow content to wrap properly inside a hexagonal cell, in the absence of browser support for the CSS shape-inside property.)

    Hex Grid

    mixins

    grid

    @mixin grid($columns: 1, $max-rows: 1, $stagger-direction: 'down', $grid-spacing: 0.5em) { ... }

    Description

    Configure hex grid layout on a container element.

    Due to current limitations of CSS Grid's implicit grid, Hextile needs to explicitly define the underlying rectangular grid from which its hex grid is derived, rather than rely on runtime auto-placement. Any member elements in excess of the product of $columns * $max-rows will be suppressed (set to display: none). It is therefore inadvisable to employ a Hextile grid when laying out items whose total count can't be easily predicted.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $columns

    Number of grid columns to pre-allocate.

    		Number1
    $max-rows

    Maximum number of rows to pre-allocate.

    		Number1
    $stagger-direction

    Direction to offset alternating columns, relative to the first.

    		Down or Up'down'
    $grid-spacing

    Gap to leave between hex cells.

    		Length or Percentage0.5em
